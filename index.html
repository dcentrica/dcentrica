Our Work

WalletMatrix

Launched as beta in Q2 2020, WalletMatrix is a Bitcoin wallet discovery and feature-comparison service that boasts a comprehensive search UI, vendor-supplied data, PGP verifiable data as well as a REST API and content-managed contextual help.

WalletMatrix is built on the Silverstripe framework for PHP which gives us powerful content management capabilities on a very flexible software development framework.
